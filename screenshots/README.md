# Screenshots

Place before/after screenshots here, using this naming convention:

- `flaw-N-before-X.png` — vulnerable behaviour (vulnerable code active)
- `flaw-N-after-X.png` — fixed behaviour (fix uncommented, original commented out)

Where `N` is 1–5 (the flaw number) and `X` is a short tag describing the shot
(e.g. `flaw-1-before-url`, `flaw-1-before-other-user-entry`,
`flaw-1-after-404`).

Each flaw should have at least one before and one after shot.
